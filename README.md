# Flex & Bison Examples

## Build
On Unix-like System:
```
mkdir build && cd build
cmake ..
make
```

On Windows, please read [instructions](doc/01-Using_Flex_and_Bison_on_Windows.md).

## References

1. John Levine, flex & bison, O'Reilly, 2009.
2. Charles N. Fischer et al, Crafting a Compiler, Pearson, 2009.
